------- PL -------

Aplikacja Budżet_2022

Prosta aplikacja pozwalająca na zapisywanie i zliczanie wydatków wraz z wpływami.

1 - Stwórz nowy okres rozliczeniowy, podając miesiąc lub miesiąc i rok (dla porządku). Następnie dodaj przychód.

Kolejnym krokiem jest wybranie ilości pozycji do wprowadzenia oraz wpisanie nazwy i ceny.

Gdy nazwa powtarza się, program dodaje wartość do bieżącej nazwy (np.: Jedzenie 10, Jedzenie 20 to Jedzenie 30).

2 - Znajdź okres rozliczeniowy

Pozwala dodać kolejny przychód, wydatki lub wyświetlić raport.

3 - Wyjście z programu


------- ENG -------

Application Budget_2022

A simple application that allows you to record and count expenses along with receipts.

1 - Create a new accounting period, specifying the month or month and year (for order). Then add the revenue.

The next step is to select the number of items to enter and enter the name and price.

When the name repeats, the program adds the value to the current name (for example: Food 10, Food 20 is Food 30).

2 - Find accounting period

Allows you to add another revenue, expense, or view a report.

3 - Exit the program
